# YandexHome - Интеграция Yandex Smart Home

![YandexHome Icon](static/YandexHome.png)

Интеграция с Yandex Smart Home (Яндекс Алиса) для управления устройствами умного дома через голосовые команды и приложение Яндекса.

## Описание

Модуль `YandexHome` предоставляет интеграцию с платформой Yandex Smart Home для платформы osysHome. Он обеспечивает голосовое управление устройствами умного дома через голосового помощника Яндекс Алиса.

## Основные возможности

- ✅ **Голосовое управление**: Управление устройствами через Яндекс Алису
- ✅ **Типы устройств**: Поддержка различных типов устройств (свет, переключатели, датчики и т.д.)
- ✅ **OAuth аутентификация**: Безопасная аутентификация на основе OAuth
- ✅ **Обнаружение устройств**: Автоматическое обнаружение и регистрация устройств
- ✅ **Сопоставление свойств**: Сопоставление свойств osysHome с возможностями устройств Yandex
- ✅ **Синхронизация состояния**: Синхронизация состояния в реальном времени

## Панель администратора

Модуль предоставляет административный интерфейс для:
- Просмотра устройств Yandex
- Настройки параметров устройств
- Управления возможностями устройств
- Связывания устройств с объектами osysHome

## Требования для настройки

- Доменное имя
- SSL сертификат (рекомендуется Let's Encrypt)
- Конфигурация навыка Яндекс Диалоги

## Конфигурация

- **User ID**: Идентификатор пользователя Yandex
- **User Password**: Пароль пользователя для аутентификации
- **Client ID**: Client ID навыка Яндекс Диалоги
- **Client Secret**: Client secret навыка Яндекс Диалоги
- **Client Key**: Ключ клиента для API
- **Skill ID**: ID навыка Яндекс Диалоги

## Инструкции по настройке

1. Посетите https://dialogs.yandex.ru/
2. Создайте новый навык -> Создать диалог -> Умный дом
3. Установите endpoint URL: `https://your-domain/YandexHome`
4. Настройте OAuth URLs:
   - Авторизация: `https://your-domain/YandexHome/auth/`
   - Токен: `https://your-domain/YandexHome/token/`
5. Введите учетные данные в настройках модуля
6. Протестируйте подключение в панели тестирования Яндекс Диалоги

## Использование

### Добавление устройства

1. Перейдите в модуль YandexHome
2. Нажмите "Add Device"
3. Выберите объект osysHome
4. Настройте тип устройства и возможности
5. Устройство доступно в приложении Yandex

## Технические детали

- **Протокол**: API Yandex Smart Home
- **Аутентификация**: OAuth 2.0
- **Типы устройств**: Свет, переключатели, датчики, термостаты и т.д.
- **Возможности**: вкл/выкл, яркость, цвет, температура и т.д.
