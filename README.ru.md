# YandexHome - Интеграция Yandex Smart Home

![YandexHome Icon](static/YandexHome.png)

Интеграция с Yandex Smart Home (Яндекс Алиса) для управления устройствами умного дома через голосовые команды и приложение Яндекса.

## Описание

Модуль `YandexHome` предоставляет интеграцию с платформой Yandex Smart Home для osysHome. Он обеспечивает голосовое управление устройствами через Алису, сопоставление свойств объектов osysHome с возможностями устройств Яндекса и синхронизацию состояния в реальном времени.

## Основные возможности

- **Голосовое управление** через Яндекс Алису
- **Типы устройств**: свет, переключатели, датчики, термостаты и др.
- **OAuth-авторизация** для связки аккаунтов
- **Обнаружение устройств** и уведомления об изменении состояния
- **Привязка** свойств osysHome к capabilities устройств Yandex

## Панель администратора

- Просмотр и группировка устройств по комнатам
- Настройка типов, capabilities и привязок к объектам
- Параметры навыка и OAuth-токенов

## Технические детали

- **Протокол**: Yandex Smart Home API
- **Аутентификация**: OAuth 2.0
- **Категория**: App
- **Действия**: `search`
- **Версия**: 0.2
